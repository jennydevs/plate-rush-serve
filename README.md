# Plate, Rush, Serve!

These are the files for [Plate, Rush, Serve!](https://jennydevs.itch.io/plate-rush-serve), a fast-paced restaurant game with singleplayer and local multiplayer modes (single computer for now). It offers time-limited games, and endless mode. You can compete with another player for high scores!

You can read about my process here in my [first devlog about this game](https://jennyton88.github.io/devlogs/log/2024_10_23_0).
